GMResourceFork

Discussion

A utility class to construct raw resource fork data. In OS 10.4, the ResourceFork for a file may be present in an AppleDouble (._) file that is associated with the file. In 10.5+, the ResourceFork is present in the com.apple.ResourceFork extended attribute on a file.
