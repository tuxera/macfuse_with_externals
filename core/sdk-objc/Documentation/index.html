MacFUSE Framework

An Objective-C API for using MacFUSE to create file systems.

MacFUSE.framework is an Objective-C API that uses MacFUSE to enable you to easily create a file system in user space.

The best way to get started with MacFUSE.framework is to look at some example file systems. See HelloFS, LoopbackFS, and other example file systems found here.
